# 맨몸 서킷 · 홈트 타이머 🏋️

장비 없이 어디서나 할 수 있는 맨몸 서킷 운동 웹앱.
단일 HTML 파일로 동작하며, 인터넷 없이도 100% 작동합니다.

**▶ 라이브:** https://suicide7803.github.io/bodyweight-circuit/

## 기능
- **프리셋 4종** — 초보 전신 · 빠른 HIIT · 코어 집중 · 하체 집중
- **운동 21종** — 하체 / 상체 / 코어 / 전신, 검색 · 카테고리 필터
- 각 동작 **SVG 일러스트 + 한글 수행법**
- **인터벌 타이머** — 준비 → 운동 → 휴식, 원형 게이지 + 비프음
- 운동 시간 · 휴식 · 라운드 직접 조절
- 일시정지 / 이전 / 스킵, 다음 동작 미리보기
- 운동 중 화면 꺼짐 방지(Wake Lock), 완료 요약

## 실행
`index.html` 파일을 브라우저로 열면 끝. 빌드 과정 없음.

## 배포 (GitHub Pages)
1. 이 저장소를 GitHub에 푸시
2. **Settings → Pages → Branch: `main` / `(root)`** 선택 후 Save
3. 잠시 뒤 `https://<사용자명>.github.io/<저장소명>/` 로 접속

## 기술
순수 HTML + CSS + JavaScript (의존성 없음). 단일 파일.
